# 🛡️ PassGuard Pro — Advanced Password Intelligence

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Vite](https://img.shields.io/badge/Frontend-Vite-646CFF?logo=vite)
![Security](https://img.shields.io/badge/Security-Advanced-brightgreen)

**PassGuard Pro** is a premium, high-performance password security analyzer and generator. It uses cryptographic entropy analysis and pattern detection to help users create unbreakable digital identities.

---

## ✨ Key Features

- **🔍 Deep Analysis**: Real-time entropy calculation (bits), length verification, and charset size detection.
- **⚡ Pro Generator**: Create cryptographically strong passwords or memorable passphrases with custom rules.
- **💀 Cracker Simulator**: Educational simulation of brute-force and dictionary attacks to visualize vulnerability.
- **🛡️ Wisdom Vault**: Expert security tips to help you stay ahead of modern cyber threats.
- **🎨 Premium UI**: Stunning glassmorphism design with smooth animations and responsive layouts.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Version 16 or higher)

### Installation & Local Run
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yogaraj02/Password-Analyser-Project.git
   cd Password-Analyser-Project
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Launch the development server:**
   ```bash
   npm run dev
   ```
4. Open the provided Localhost URL in your browser!

## 🔒 Security Principles

PassGuard Pro analyzes passwords based on:
1. **Entropy**: The mathematical measure of randomness.
2. **Patterns**: Detection of keyboard walks, sequences, and common dictionary words.
3. **Composition**: Balance of uppercase, lowercase, numbers, and special characters.
